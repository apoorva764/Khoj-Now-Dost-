Project Name: Khoj
Tagline: Your AI-Powered Study Planner and Bookstore Assistant

🔧 Technologies Used
Python – The main programming language.
Streamlit – Used to build the web-based app quickly and easily.
Google Gemini Pro API – For generating smart responses using AI.
PyPDF2 – To read and extract text from uploaded PDF files.
dotenv – To manage API keys securely.

What Khoj Does 

Khoj is a dual-purpose web app designed to help:
Students plan their studies efficiently based on syllabus, exam dates, and available time.
Users ask about bookstore inventory and get intelligent recommendations.
Analyze PDF study materials and suggest key points, revision tips, and questions.

Main Features You Can Talk About

1. Study Planner Mode
User enters syllabus, exam date, and daily available hours.
Khoj uses AI to create a day-by-day study schedule.
Schedule includes study sessions, breaks, revision time, and milestones.

2. Bookstore Assistant Mode
Khoj shows a sample book inventory (fiction, non-fiction, textbooks).
User types a query like “Suggest a good Python book”.
AI checks the inventory and gives a helpful response.

3. PDF Study Material Analyzer
Users can upload a PDF file (like class notes).
khoj reads the content and gives:
Key topics
Revision plan
Memory tips
Practice questions

What Makes Khoj Special?
Combines AI (Gemini Pro) with education and inventory management.
Offers real help for students in saving time and learning smarter.
Designed using lightweight tools like Streamlit, so it runs quickly in browsers.
Won 1st Runner Up in a Hackathon – shows practical value and team effort.
